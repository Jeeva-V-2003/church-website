# LHMM Website - Project Overview

## What Has Been Created

A complete, professional multi-page religious website for Bible Standard Ministries with:

### ✅ Backend (NestJS + MongoDB)
- RESTful API with 3 main modules
- Daily Manna management system
- Magazine publication system
- Events management system
- TypeScript for type safety
- MongoDB schemas with Mongoose
- CORS enabled for frontend communication

### ✅ Frontend (React 18)
- 8 Complete Pages:
  1. Home - Hero section with features
  2. Daily Manna - Full devotional content
  3. Magazines - Grid with covers
  4. Magazine Detail - Individual magazine view
  5. Events - Tabbed upcoming/past events
  6. Event Detail - Individual event view
  7. About - Mission, values, history
  8. Contact - Form and contact info

### ✅ Design Features
- Elegant brown/sand religious color scheme
- Glass morphism effects
- Smooth Framer Motion animations
- Floating bubble backgrounds
- Responsive mobile-first design
- Professional typography
- Hover effects and transitions

### ✅ Components
- Navbar with scroll effects
- Footer with social links
- Reusable page layouts
- Card components
- Form components

## Technology Stack

**Backend:**
- NestJS (Node.js framework)
- MongoDB + Mongoose
- TypeScript
- Class Validator

**Frontend:**
- React 18
- React Router v6 (multi-page)
- Framer Motion
- Axios
- React Icons
- Custom CSS3

## Next Steps

1. **Install Dependencies:**
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```

2. **Start MongoDB:**
   ```bash
   sudo systemctl start mongod
   ```

3. **Run Backend:**
   ```bash
   cd backend && npm run start:dev
   ```

4. **Run Frontend:**
   ```bash
   cd frontend && npm start
   ```

5. **Add Sample Data:**
   Use the API endpoints to populate with content
