# Bible Standard Ministries Website

Official website for Bible Standard Ministries - Laymen's Home Missionary Movement (LHMM)

## About LHMM

The Laymen's Home Missionary Movement is a Christian organization dedicated to spreading Bible truth through literature, conventions, and missionary work worldwide.

## Features

- **Daily Scripture Study** - Daily Bible readings and devotionals
- **Publications** - Access to Bible Standard magazines and literature
- **Convention Information** - Details about upcoming Bible conventions
- **Missionary Work** - Information about global missionary activities
- **Bible Studies** - Online Bible study resources

## Quick Start

```bash
git clone https://github.com/Jeeva-V-2003/church-website.git
cd church-website/mern
npm run setup
npm run dev
```

## Tech Stack

- **Frontend**: React, CSS3
- **Backend**: NestJS, MongoDB
- **Features**: Responsive design, Content management

## Development

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

Frontend: http://localhost:3000  
Backend: http://localhost:5000
